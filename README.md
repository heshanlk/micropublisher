# Micropublisher for Drupal

Allows users to easily attach various kinds of content (links, photos, videos, etc) in a single content field so you can create a stream like on Facebook

### Currently supported: 

* Bliptv
* Flickr
* Hulu
* Link
* Image upload
* Metacafe
* Slideshare
* Twitpic
* Twitter
* Viddler
* Vimeo
* Wikipedia
* Yfrog
* Youtube
